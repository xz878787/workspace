## 行内/块级
div, ul 
span 等行内
- 块级 block 盒子
 - 可以设置宽高
 - 独占一行
- 行内 inline
  - 不可以设置宽高
  - 也不会把兄弟挤下去 
- display 属性
   flex 开启弹性格式上下文
   inline-block 行内块级
    - 可以设置宽高
    - 不会把兄弟挤下去
  浏览器默认块级/行内 ->  display手动切换 inline/block-> 
  格式化上下文 (flex/inline-block/grid)
     inline-block 默认有一个天坑
     默认空格符会占据一定的大小 \你\r
