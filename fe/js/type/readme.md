# JS 有多少种数据类型？
八种数据类型
根据ECMA262规定
primitive 原始数据类型有6种，...，
其中numeric 再分为number和新增的bigint,还有一种对象

## 简单数据类型 ， 
原始数据类型(primitive types)
- Number 数值
- Boolean 布尔值
- String 字符串
- null
    - 用于表示一个有意设置为空的对象引用。
    - 表示某处应该有个值， 不过目前没有(空的值)
    - 可以用于清空一个变量的值， 释放内存， 垃圾回收
    
- undefined 
    表示一个未初始化 或不存在的变量值 
    - 当申明一个变量， 但是没有赋值时
    - 某个对象的属性不存在
    - 函数没有返回值时，
    - 访问不存在的数组索引 
 es6 以前 js 只有6种类型
 ### es6 新增数据类型
 - symbol 唯一
 - bigint 大数      

## 复杂数据类型
引用数据类型(内存分配)、
- object

   栈顶元素位置 ，指针偏移量就可以切换上下文的执行 快速、稳定、扩展
