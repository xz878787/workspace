# JS 有多少种数据类型？
## 简单数据类型 ， 
原始数据类型
- Number 数值
- Boolean 布尔值
- String 字符串
- null
    - 用于表示一个有意设置为空的对象引用。
    - 表示某处应该有个值， 不过目前没有(空的值)
    - 可以用于清空一个变量的值， 释放内存， 垃圾回收
    
- undefined
