# 快速排序

数组飘絮
- 冒泡 两两相邻位置 比较
- 选择排序 选择最小的元素 放到当前位置
- 插入排序 从当前位置开始 与前面的元素比较 直到找到合适的位置
O(n^2)

## 快速 O(nlogn)
- pivot 基准值
- 比他小的放左边
- 比他大的放右边
双指针交换

- 递归 对左边和右边的数组分别再做

快排 (quick sort)  是一种基于 **分治策略** 的排序算法，运行高效(O(nlogn))  (平均情况), 应用广泛
分而治之

## 递归，分治的区别
递归是代码实现方式(函数自身调用)
自顶向下 缩小问题的规模
分治是算法思想  ，绝大多用递归实现
分 把大问题分成互相独立的小问题
治 解决小问题
合 合并小问题的结果

pivot 左边右边

## 总结
- 快的原因 
pivot 基准值O(logn) + 原地交换O(n) 双指针
