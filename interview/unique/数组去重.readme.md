    ## 多种解法
    - 双循环去重法
   - indexOf 去重法
     - 数组下标，原来数组index比较
       -当前项，和res数组里比较-1 留下，
       - 先排序，相邻比较， 当前项和前一项
        时间复杂度是O(nlogn)
        缺点是数组的稳定性被破坏了
        - hash table 对象自面量属性去重
        数据结构的一种，
        优点是O(n);
        空间换时间 缺点 
        数据结构解决特定问题的数据存储结构和特定行为
        - 专有数据结构 Set
        es6+
        不重复的数据容器,不是数组
        - es6 新增数据结构
