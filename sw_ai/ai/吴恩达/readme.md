         ## prompt 设计过程
         (Goal)
         -创建一个网页来帮我写生日贺卡,
           当我提供一个人的姓名、年龄、爱好、特征等信息时，他应该会给我返回一条有趣的消息
           如果不满意，接着prompt，沉静在Vibe coding 氛围中
           -加上节日风格的标准和配色，让他更好看些
           -把卡片放在右侧展示，让它看起来像生日贺卡的内页
            -在页面的顶部加一个有趣的标题。另外，不要替换掉旧卡片，而是保留他们并且叠放在新卡片的下方。Features 特点
            当我们用AI写代码时，不是随意去写prompt的，通常从一组基本指令开始
            Prompt 基本块
            Goal +Input+ Output+ Layout+ Features
            
创建一个网页来帮我写生日卡片。
当我输入一个人的姓名、年龄、爱好后，他应该返回一条有趣的消息。
使用喜庆的标题和配色。
把贺卡显示在右侧，并且让它看起来像生日贺卡的内页。
在页面顶部添加一个有趣的标准。
另外，不要替换旧的贺卡，而是保留他们的显示状态，把他们堆叠在新贺卡的下方。

AI擅长执行准确清晰的指令
