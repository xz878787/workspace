## 知识库
- 海量的训练数据
- 瓶颈是什么？
  - 过时的，最新的新闻是没有的
  - 个人、企业私有的一些知识没有
  如何给llm 提供它没有的知识呢?
  存放各类专业资料、供 AI 随时调取参考的素材仓库。

  ## 搭建智能体过程
  - 新建 coze agent
  - prompt 方式 人设，功能描述
  - 人设：你是一个智能客服机器人，你收到用户的提问后，需要调用知识库 去查询内容，将查询到的信息返回给用户。
注意：
1.你不要编造内容给用户，只能将知识库的信息返回给用户。
2.如果问题在知识库查询不到，那么提醒用户："对不起，目前您问的问题，我们还无法回答。加微信ysw123123"

  ## Prompt 规则
  - 给他一个人设
- 告诉他能干什么，不能干什么
