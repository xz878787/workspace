## 数据分类
- 训练集 train datasets
  学生用的教材 吸收知识
- 验证集 validate datasets
  课后作业,帮助学生查缺补漏
- 测试集 test datasets
  考试,测试学生的学习效果
  
  训练集目的让模型去查找数据的一个规律。
  验证集就是**频繁**的去验证我的模型训练好坏，来帮助我们去调节一些参数。
  测试集就相当于去验证我训练好的模型，他在没有看到的数据上的一个**泛化**能力。

模型不是死记训练数据， 能对全新没见过的陌生输入，做出正确、贴合逻辑的回答与判断能力

  ## 交叉验证
  如果静态的 8：1：1 划分， 容易让模型偏科， 交叉验证让模型换个角度看数据。

  - k折交叉验证
  把数据集分为k份， 每次留一份去做测试， 其余的都拿去做训练，k次循环后
  所有的数据都被训练且被测试了。 k= 10
