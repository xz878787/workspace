# 蒸馏

- 什么是蒸馏?
小模型公司怎么去学大模型的本事?
2025年， DeepSeek 花了600万美金训练成本， 做出了和OpenAI 可以扳手腕的模型。
是抄捷径。

巨量的向OpenAI 发请求， 吸星大法

## 蒸馏何谓?

化学术语， 
把混合液体加热， 让某些部分变成了蒸汽挥发，然后再去冷却收集，提取精华，弃其糟粕。

大模型蒸馏的核心思路是一样的。
就是把llm 里的精华知识，提取出来， 灌进小模型。
大模型叫老师模型，
小模型叫学生模型。
提取的过程叫知识蒸馏
虚竹 无崖子的传毕生功力， 就是知识蒸馏。


- 向大模型发送请求 得到答案
题目和标准答案 背答案
比如 记住这个图是猫， 对或错，
模型死记答案， 这个叫**硬标签**

不够的，

小模型学的不是答案， 而是思维方式。
大模型在回答问题的时候，其实不是给你一个答案，而是一组概率

软标签 不是一个答案， 它是一个概率分布，比如，猫的概率80%， 狗的概率10%， 鸡的概率..

软标签比硬标签更有信息量， 它包含llm 对这道题的理解方式。

答案不只是猫那么简单，而是猫和狗有相似之处， 猫和老虎比较像，但是没狗那么像。这种软标签类比感，关联感是大模型积累了海量的数据后，形成的我们定义为暗知识。

那蒸馏， 就是让小模型去模仿这一套的概率分布， 而不只是去背标准答案，而是思维方式。

打个比方， 我们要培养一个厨师新人，第一种方案，给他菜谱 硬标签。
新人只学会了步骤，没学到别的。
第二种方法，让米其林厨师亲自示范，新人在旁边看，大厨怎么去判断火候
怎么去凭感觉
